# 📚 Library Management System

A modern, full-stack library management solution built with React and Node.js

## ✨ Features

- 🔐 Secure Authentication System
- 📱 Responsive Design
- 📖 Book Management
- 👥 User Management
- 📧 Email Notifications
- 🎨 Beautiful UI with Tailwind CSS

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js
- 🎨 Tailwind CSS
- 📡 Axios
- 🔄 React Router
- 🍞 React Hot Toast

### Backend
- 🚀 Node.js
- 🛣️ Express.js
- 🗃️ MongoDB
- 📧 Nodemailer
- 🔒 JWT Authentication
